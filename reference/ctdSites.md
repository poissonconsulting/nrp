# A site table for Sea-bird CTD data

A site table for Sea-bird CTD data

## Usage

``` r
ctdSites
```

## Format

A tbl data frame:

- SiteID:

  Site Identifyer

- SiteNumber:

  Site Number

- SiteName:

  Common Name or description the Site

- BasinArm:

  Which arm or Basin it is located in

- Depth:

  The depth of the Seabird

- geometry:

  Coordinates for the location of the site
