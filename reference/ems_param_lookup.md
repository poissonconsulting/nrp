# A lookup table for for selecting the desired columns from EMS

A lookup table for for selecting the desired columns from EMS

## Usage

``` r
ems_param_lookup
```

## Format

A tbl data frame:

- PARAMETER_CODE:

  PARAMETER_CODE

- PARAMETER:

  PARAMETER

- ANALYTICAL_METHOD:

  ANALYTICAL_METHOD

- chem.make.names:

  chem.make.names

- chem.names:

  chem.names

- Comment:

  Comment
