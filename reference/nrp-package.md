# nrp: Kootenay Lake Nutrient Restoration Program Data Loading

Imports CTD, EMS, Zooplankton, and Mysid data for the Kootenay Lake
Nutrient Restoration Program (NRP) and provides functionality for
uploading and downloading with an SQLite database.
